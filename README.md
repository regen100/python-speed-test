python-speed-test
=================

Speed test for pure Python, numpy, weave, cython and cython (OpenMP)

Results
=======

* Intel Core i7 860 2.80GHz
* Windows 7 64bit
* Python 2.7.6
* Numpy 1.8.0
* Scipy  0.13.3
* Cython 0.20

| Type            | Time (ms) |
| --------------- | ---------:|
| Pure Python     | 48300     |
| Numpy           | 1860      |
| Weave           | 785       |
| Cython          | 765       |
| Cython (OpenMP) | 189       |
